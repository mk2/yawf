# @yawf/yawf-cli

## What's this?

- Yet Another Web Framework
- Simple and Easy
- Express.js based
- ESModule oriented

## How to start?

### Let's start with new yawf project!

```
# ... installed Node.js( > v.10) and npm
npm i -g @yawf/yawf-cli
mkdir project_dir; cd project_dir
yawf g app
npm start
```
