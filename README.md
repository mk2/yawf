# yawf - Yet Another Web Framework

- Simple and Easy
- ESModule oriented
- Inspired By Sails.js

# How tu run

## Ready for development
```
npm i
npm run packageInstall
```

## Run demo project
```
cd ./packages/yawf-demo-project
node app.js
```

## Make project by @yawf/cli
```
npm i -g @yawf/yawf-cli
mkdir project_dir; cd project_dir
yawf generate app
```
